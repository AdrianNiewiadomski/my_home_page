Methods of generating vertical cross sections of 3D objects

Bachelor of Engineering thesis | Author: Adrian Niewiadomski | Supervisor: Krzysztof Wr√≥bel PhD


As part of the thesis, a theoretical solution was proposed and the implementation of the algorithms was presented solving the following problems occurring in computer graphics:

  • create correctly displayed vertical sections of 3D objects,

  • display the faces of a 3D object by correcting the order of vertices in the faces (and thus normal vectors).

The application was written in JavaScript using the Three.js library. The backend has been written in Python (Flask).
